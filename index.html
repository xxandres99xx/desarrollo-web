Análisis del problema

Reconocimiento del Problema

La firma de abogados "Te Apoyamos SAS" desea ampliar su cobertura de servicios a través de una herramienta digital que le permita darse a conocer en el mercado global. Para ello, necesita una aplicación web que cumpla con varios propósitos clave:

  • Brindar información sobre la firma y sus servicios.
  • Permitir el registro de hojas de vida de profesionales interesados en trabajar con la firma.
  • Gestionar los procesos legales que adelantan los profesionales.
  • Garantizar la seguridad de la información y el respaldo de los datos.

Objetivos de la Aplicación Web

  • Diseñar e implementar una plataforma web intuitiva y segura, que facilite la gestión de información sobre la firma, mostrando sus servicios, profesionales y noticias relevantes que puedan atraer clientes.
  • Proveer un sistema de registro y evaluación de hojas de vida, que permita realizar un pequeño formulario donde pongan información relevante, que puede facilitar la evaluación, al tener en cuenta requisitos específicos.
  • Implementar una herramienta de administración de procesos legales, para estar al tanto de los casos que lleven los profesionales de la firma y su estado en tiempo real.
  • Garantizar la seguridad de la información mediante mecanismos de autenticación y cifrado de datos.

Características de la aplicación

  • La aplicación debe contar con un módulo informativo que contenga, una página principal con información sobre la firma, trayectoria y reconocimientos, descripción detallada de los servicios que ofertan, noticias de interés sobre casos actuales o casos importantes a cargo de la firma, además de una sección de contacto, donde se encuentre la información relevante para establecer conexión con los clientes.
  • Un segundo módulo para registro de profesionales que contenga un formulario de inscripción, con preguntas clave que faciliten el proceso de evaluación y un botón para cargar la hoja de vida, una base de datos que almacene y gestione los perfiles de los profesionales aspirantes.
  • Un tercer Módulo para la gestión de procesos legales que contenga los registros de clientes y tipos de procesos legales, seguimiento del avance de cada proceso, además de control de acceso para cada profesional a la información más importante del caso.

Requerimientos Técnicos

  • Desarrollo en un entorno web moderno y seguro.
  • Base de datos para gestión de usuarios, hojas de vida y procesos.
  • Implementación de autenticación y roles de usuario.
  • Respaldos periódicos de información.

Seguridad y Respaldo de Datos

  • Encriptación de datos sensibles.
  • Uso de HTTPS para la comunicación segura.
  • Implementación de roles y permisos de acceso.
  • Copias de seguridad automatizadas.

Frameworks

son conjuntos de herramientas, librerías y convenciones que proporcionan una base estructurada para el desarrollo de aplicaciones. Ayudan a los desarrolladores a trabajar de manera más eficiente al ofrecer componentes predefinidos, patrones de diseño y funcionalidades comunes, lo que permite centrarse en la lógica específica de la aplicación en lugar de construir todo desde cero. Algunas de estas herrmientas son:

Angular

Framework de JavaScript basado en TypeScript, usa el patron MVC, que organiza el código y mejora la usabilidad, compatible con Bootstrap.

Bootstrap:

Es un framework de CSS y JavaScript, para diseño responsivo, cuenta con componentes prediseñados que mejoran la experiencia del usuario, su sistema de cuadricula lo hace flexible, para que pueda adaptarse a diferentes tamaños de pantalla.

React:

Es un framework de la biblioteca de JavaScript, que facilita la creación de interfaces interactivas y permite integrarse con CSS como Bootstrap.
